// 04 — ciberseguridad

Criptografía &
Seguridad

La ciberseguridad matemática me apasiona especialmente. Mi trabajo con curvas elípticas demuestra cómo la teoría de números pura se convierte en la base de los sistemas de seguridad que usas todos los días.

🔐
Criptografía de Curvas Elípticas (ECC)
Sistema de cifrado basado en la estructura algebraica de curvas elípticas sobre campos finitos. Misma seguridad que RSA con claves más cortas.
∿
Curvas sobre Z₃
Operaciones de grupo sobre curvas elípticas en Z₃, desde cero, sin ninguna librería externa. Pura matemática aplicada.
🔑
Intercambio de Llaves
Protocolo Diffie-Hellman sobre grupos de puntos en curvas elípticas para canales seguros sin transmitir la clave privada.
📊
Hash & Integridad
Funciones hash criptográficas para verificación de integridad. Fundamentos de SHA-256 y su relación con estructuras matemáticas.

# Curva elíptica sobre Z₃: y² = x³ + ax + b (mod p) # Implementación sin librerías — Juan Pablo Rodríguez García def point_add(P, Q, a, p): if P == (None, None): return Q if Q == (None, None): return P x1, y1 = P; x2, y2 = Q if x1 == x2 and (y1 + y2) % p == 0: return (None, None) if P == Q: m = (3 * x1**2 + a) * pow(2 * y1, p-2, p) % p else: m = (y2-y1) * pow(x2-x1, p-2, p) % p x3 = (m**2 - x1 - x2) % p y3 = (m * (x1-x3) - y1) % p return (x3, y3) R = point_add((3,10), (9,7), a=2, p=97)
